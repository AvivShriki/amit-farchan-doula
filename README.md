# עמית פרחן — דולה מוסמכת

דף נחיתה לעמית פרחן, דולה מוסמכת המתמחה בייעוץ הנקה, לצד ליווי הריון ולידה.

## מבנה הפרויקט

- `index.html` — עמוד הבית (הירו, שירותים, תהליך עבודה, מי אני, המלצות, שאלות נפוצות, צור קשר)
- `pregnancy.html` — ליווי הריון
- `birth.html` — ליווי לידה
- `breastfeeding.html` — ייעוץ הנקה
- `css/style.css` — מערכת עיצוב משותפת (טוקנים, רכיבים, אנימציות, רספונסיביות)
- `js/main.js` — תפריט מובייל, קרוסלת המלצות, אקורדיון שאלות נפוצות, אנימציית סקרול (מכבדת prefers-reduced-motion)
- `landing_page_prompt_doula_1.md` — תבנית הבריף המקורית ששימשה לבניית התוכן
- `assets/photos/` — תמונות אמיתיות של עמית שנבחרו לשימוש באתר (מתוך תיקיית `amit photos /` המקורית), כולל `logo-mark.png` — הלוגו האמיתי (חתוך ורקע שקוף)
- `amit photos /` — כל התמונות הגולמיות שהעלתה הלקוחה (ארכיון מקור, לא כולן בשימוש באתר, לא נדחף לריפו הציבורי)

## הרצה מקומית

```
python3 -m http.server 4500
```

ואז פתחו `http://localhost:4500`.

## פלטת הצבעים

צבעי המבטא (`--color-olive` זית כהה, `--color-taupe` טופ חם) נשאבו ישירות
מחומרי שיווק קיימים של הלקוחה (`amit photos /IMG_1987.JPG`) לפי בקשתה
המפורשת לשמור על הצבעים שמייצגים אותה — לא צבעים שהומצאו.
