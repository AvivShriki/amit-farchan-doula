# עמית פרחן — דולה מוסמכת

דף נחיתה לעמית פרחן, דולה מוסמכת המתמחה בייעוץ הנקה, לצד ליווי הריון ולידה.

## מבנה הפרויקט

- `index.html` — עמוד הבית (הירו, שירותים, מי אני, המלצות, שאלות נפוצות, צור קשר)
- `pregnancy.html` — ליווי הריון
- `birth.html` — ליווי לידה
- `breastfeeding.html` — ייעוץ הנקה
- `css/style.css` — מערכת עיצוב משותפת (טוקנים, רכיבים, רספונסיביות)
- `js/main.js` — תפריט מובייל, קרוסלת המלצות, אקורדיון שאלות נפוצות

## הרצה מקומית

```
python3 -m http.server 4500
```

ואז פתחו `http://localhost:4500`.

## הערות

- כל התמונות הן תמונות placeholder מ-Unsplash (אישה ניטרלית, לא עמית) — יש להחליף בתמונות האמיתיות של עמית בהמשך.
- מספרי הוואטסאפ וכתובת המייל בכפתורי יצירת הקשר הם placeholder לדוגמה — יש לעדכן לפרטי הקשר האמיתיים.
