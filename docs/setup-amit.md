# הוראות לעמית — ניהול האתר שלך בעצמך 🤍

היי עמית! האתר שלך עובר להיות בשליטה מלאה שלך: האחסון על שמך, הדומיין על שמך,
ואת יכולה לערוך טקסטים ותמונות בעצמך — בלי לדעת כלום על קוד.
המסמך הזה מסביר מה צריך לעשות פעם אחת (הקמה), ואיך עורכים ביום־יום.

---

## חלק א' — הקמה חד־פעמית (כ־20 דקות, יחד עם אביב)

### 1. פתיחת חשבון GitHub (חינם)
זה המקום שבו שמור "קוד המקור" של האתר.

1. נכנסים לאתר ההרשמה: github.com/signup
2. נרשמים עם האימייל האישי שלך ובוחרים סיסמה.
3. שומרים את שם המשתמש והסיסמה במקום בטוח — זה החשבון שאיתו תתחברי לעריכת האתר.

### 2. פתיחת חשבון Netlify (חינם)
זה האחסון — השרת שמציג את האתר לעולם.

1. נכנסים לאתר ההרשמה: app.netlify.com/signup
2. בוחרים להירשם עם GitHub — כפתור "Sign up with GitHub" — ומאשרים.
   (כך שני החשבונות מחוברים ואין סיסמה נוספת לזכור.)

### 3. חיבור האתר (אביב עושה איתך)
1. אביב מעביר אליך את מאגר הקוד של האתר, או מוסיף אותך כשותפה.
2. בתוך Netlify לוחצים: Add new site ← Import an existing project ← GitHub
   ובוחרים את המאגר של האתר.
3. Netlify מזהה לבד את הגדרות הבנייה (הקובץ netlify.toml כבר בפנים) — לוחצים Deploy.
4. תוך דקה־שתיים האתר באוויר בכתובת זמנית של Netlify.

### 4. חיבור הדומיין שלך
1. קונים דומיין על שמך אצל רשם ישראלי (אביב ימליץ איפה), בכרטיס שלך,
   עם חידוש אוטומטי — כדי שהאתר לא ייפול בטעות.
2. בתוך Netlify: Domain settings ← Add custom domain ← מקלידים את הדומיין.
3. אצל רשם הדומיין מגדירים את הרשומות ש־Netlify מציג (אביב יעזור — זה 5 דקות).
4. ההצפנה (המנעול הירוק) מופעלת אוטומטית, בחינם.

## חלק ב' — עריכת האתר ביום־יום

### נכנסים לפאנל
1. פותחים בדפדפן את כתובת האתר שלך עם התוספת admin/ בסוף.
   לדוגמה: `www.your-domain.co.il/admin/`
2. לוחצים "Sign In with GitHub" ומתחברים עם החשבון שפתחת.

### עורכים
1. בצד רואים את עמודי האתר: הגדרות כלליות, עמוד הבית, וארבעת עמודי השירותים.
2. בוחרים עמוד — וכל הטקסטים והתמונות שלו מופיעים כשדות ברורים בעברית.
3. משנים מה שרוצים: מתקנים ניסוח, מחליפות תמונה (כפתור בחירת תמונה — אפשר
   להעלות חדשה מהמחשב או מהטלפון), מוסיפות המלצה חדשה.
4. לוחצים שמירה (Save) למעלה.
5. תוך דקה־שתיים האתר מתעדכן לבד. זהו!

### דברים שכדאי לדעת
- אי אפשר "להרוס" את העיצוב — הפאנל נותן לשנות רק תוכן. העיצוב שמור.
- כל שמירה נרשמת בהיסטוריה, אז תמיד אפשר לחזור אחורה אם משהו השתבש.
- שדות שמסומנים "לא לשנות בלי אביב" (כמו קישורים) — עדיף באמת לא לגעת בהם לבד.
- תמונות: הכי טוב להעלות תמונות באיכות טובה בכיוון אנכי (כמו הקיימות),
  והפאנל כבר ידאג לשאר.
- נתקעת? כל שינוי אפשרי לביטול, ואביב תמיד זמין בוואטסאפ 🙂
